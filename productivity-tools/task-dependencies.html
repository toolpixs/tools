    Frequently Asked Questions

    What are task dependencies?

    Task dependencies define the order in which tasks must be completed. A dependent task cannot be marked complete until its parent task is completed.

    Is this tool free to use?

    Yes, ToolPix's task dependency manager is completely free with no registration required.

    How do I set a dependency?

    Select a parent task from the dependency dropdown when adding a new task. The new task will depend on the selected task.

    Are tasks saved automatically?

    Tasks and dependencies are saved in your browser's local storage and persist across sessions until cleared.
